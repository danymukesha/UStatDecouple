# DecoupleResult Class

S4 class storing results from decoupling analysis

## Slots

- `original_stat`:

  Numeric value of the original U-statistic

- `decoupled_distribution`:

  Numeric vector of decoupled statistics

- `kernel_name`:

  Character string describing the kernel used

- `method`:

  Character string describing the decoupling method

- `p_value`:

  Numeric p-value from hypothesis testing
